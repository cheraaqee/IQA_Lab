This dataset, introduced in:
Sun, Wen, Fei Zhou, and Qingmin Liao. "MDID: A multiply distorted image database
for image quality assessment." Pattern Recognition 61 (2017): 153-168.
has two folders:
1- 'distortion_images'
2- 'reference_images'
that must be pasted in this directory ( ./IQA_Lab_v0/IQA/Full_Reference/train_fr
ee/datasets/MDID/)
